Original Work Focused

As a theme in ISM, planning and scheduling a key part of being effective in achieving success. I came across an interesting management concept, the Gantt Chart, created in the 1910s. A bar chart that illustrates project schedule, I decided to use it for discipling my own workflow. I created three project avenues that needed to be accomplished for a fulfilling original work - model development, code development and code collection. Creating a task list in order of activities with each section being able to be done concurrently. Building upon the literature and research from first quarter, I questioned how I can create an innovative work that contributed not only to my own learning but also the field itself. The best way, I decided, is to create something accessible to the normal consumer. My current course of action is to make an application of ESG investing in a way that is interactive to the investor. With the tools of the internet, high-level research can be tested out in the real world. I want to focus on applying both my skills in finance and computational skills to create a web-app that allows students and the public alike to work with ESG models, which prior, may be unknown or inaccessible.
